# Hackathon Submission Checklist

Date window: February 27-28, 2026

## Required Deliverables

- [ ] 2-minute YouTube demo video recorded and uploaded.
- [ ] Public GitHub repository is accessible.
- [ ] Project description finalized (`project-description.md`).
- [ ] 4-5 slide deck exported to PDF/PPT.

## Recommended Final Validation

- [ ] `npm run typecheck`
- [ ] `npm run build`
- [ ] API running at `http://localhost:3000/api/v1`
- [ ] `npm run smoke:demo` passes with seeded users

## Demo Day Prep

- [ ] Use `video-script-2min.md` for recording flow.
- [ ] Use `slides-outline.md` as speaker order.
- [ ] Keep one student and one professor account pre-logged for quick switching.
